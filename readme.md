
# React Hooks Study

Aplicação desenvolvida como objeto de estudo dos **Hooks do React**.

## Objetivo

Este projeto serve como:
- 📚 Documentação para futuras consultas
- 🔍 Base para aprofundamento em React
- 🎯 Referência prática dos principais hooks

## Como usar

Consulte os exemplos de implementação para entender melhor cada hook do React.

---


## Setup e Instalação

### Clonar o repositório

```bash
git clone <repository-url>
cd hooks-inicial
```

### Instalar dependências

```bash
npm install
```

### Executar a aplicação

```bash
npm start
```

A aplicação abrirá em `http://localhost:3000`
